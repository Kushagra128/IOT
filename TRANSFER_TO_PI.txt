================================================================================
  TRANSFER THIS ENTIRE FOLDER TO YOUR RASPBERRY PI
================================================================================

WHAT TO DO:
-----------
1. Copy this entire "meeting-transcriber" folder to your Raspberry Pi
   
   Methods:
   - USB Drive: Copy folder to USB, insert into Pi, copy to /home/pi/
   - SCP: scp -r meeting-transcriber pi@<pi-ip>:~/
   - Git: Clone your repository on the Pi

2. On Raspberry Pi, open terminal and run:
   
   cd ~/meeting-transcriber
   chmod +x install_rpi.sh
   ./install_rpi.sh

3. Wait 15-20 minutes for installation to complete

4. Start the backend:
   
   ./start_backend.sh

5. On your laptop, configure and start frontend:
   
   cd frontend
   ./configure_frontend.sh  (or configure_frontend.bat on Windows)
   npm install
   npm run build
   npm run serve

6. Open browser: http://localhost:3000

================================================================================
  SYSTEM REQUIREMENTS
================================================================================

Raspberry Pi:
- Raspberry Pi 3B+ or newer (Pi 4/5 recommended)
- 2GB+ RAM (4GB recommended)
- Raspberry Pi OS (Debian-based)
- USB Microphone
- Internet connection for setup
- 8GB+ free disk space

Laptop:
- Windows, Mac, or Linux
- Node.js 16+ installed
- Same WiFi network as Raspberry Pi
- Modern web browser

================================================================================
  AFTER INSTALLATION
================================================================================

Your system will work like this:

1. Backend runs on Raspberry Pi (IP: 192.168.1.xxx)
   - Listens on port 5000
   - Handles audio recording from USB microphone
   - Processes speech with Vosk
   - Generates summaries with AI

2. Frontend runs on your laptop
   - Accessible at http://localhost:3000
   - Connects to Pi backend over WiFi
   - Displays live transcription
   - Shows summaries and recordings

3. Recording Flow:
   - Click "Start Recording" on laptop
   - Speak into microphone (connected to Pi)
   - See live transcript on laptop screen
   - Click "Stop Recording"
   - View transcript and AI summary
   - Download as PDF

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Cannot connect to backend"
Solution: 
  - Check backend is running on Pi: ps aux | grep gunicorn
  - Verify Pi IP in frontend/.env.production
  - Test: curl http://<pi-ip>:5000/api/health

Problem: "No microphone detected"
Solution:
  - Check USB connection
  - Run: arecord -l
  - Test: arecord -d 5 test.wav && aplay test.wav

Problem: "Vosk model not found"
Solution:
  - Check: ls ~/meeting-transcriber/vosk-model-small-en-us-0.15/
  - Re-run: ./install_rpi.sh

Problem: "Frontend can't reach Pi"
Solution:
  - Both devices on same WiFi?
  - Get Pi IP: hostname -I
  - Update: frontend/.env.production
  - Rebuild: npm run build
