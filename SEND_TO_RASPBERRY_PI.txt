AFTER TRANSFERRING TO RASPBERRY PI:
------------------------------------

STEP 1: Install
---------------
cd ~/meeting-transcriber
chmod +x install_backend_only.sh
./install_backend_only.sh

STEP 2: Update Vosk Model Path
-------------------------------
nano iot-meeting-minutes/configs/recorder_config.yml

Change this line:
    model_path: /home/pi/vosk-model-small-en-us-0.15

To your actual Vosk model location:
    model_path: /home/pi/YOUR_VOSK_MODEL_PATH

Save: Ctrl+X, Y, Enter

STEP 3: Start Backend
----------------------
./start_backend.sh

TRANSFER METHODS:
-----------------

Method 1: USB Drive
-------------------

2. Insert USB into Raspberry Pi
3. Copy to Pi:
   cp -r /media/pi/*/meeting-transcriber ~/

Method 2: SCP (from laptop)
---------------------------
scp -r backend iot-meeting-minutes install_backend_only.sh pi@<pi-ip>:~/meeting-transcriber/

Method 3: Git
-------------
# On Raspberry Pi:
git clone <your-repo> ~/meeting-transcriber
cd ~/meeting-transcriber
